# QuadTrees_Laboratorio-3

El laboratorio aborda el desafio de gestionar 10.000 puntos de entrega en un entorno urbano, donde la eficiencia en la recuperacion de informacion geografica es critica. El problema central radica en que realizar busquedas especiales mediante fuerza bruta lo que tiene una complejidad de O(n) por lo que se implementaron Quadtrees
